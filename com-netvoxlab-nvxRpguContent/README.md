# com-netvoxlab-nvxRpguContent

Плагин для отображения контента РПГУ. Страница настроек: NVX RPGU content. 
После установки необходимо заполнить адрес удалённого Re:Doc. Указывается адрес портала РПГУ на базе Re:Doc, с которого тянутся данные, либо адрес Re:Doc с включенными плагинами Nvx.ReDoc.StateStructureServiceModule и Nvx.ReDoc.Rpgu.PortalModule.

##Доступные отображения:
* Поиск услуг/просмотр услуг
* Информация по услуге
* Информация по ведомству
* Блок авторизации ЕСИА (зависим с плагином ЕСИА)
* Популярные услуги
* Список категорий с услугами
* Категории услуг
* Жизненные ситуации
* Иерархический список ведомств
* Услуги для выбранной категории
* Информация по МФЦ
* Запись на приём
* Страница с функционалом оплаты (Юнителлер, парковки)
* Динамическая форма заявления
* Форма вложений заявления
* Просмотр информации по заявлению
* Страница личного кабинета со встроенными разделами
* Обращение
* Каталог с разделами Категории/Ведомства/Жизненные ситуации
* Отчёты шины по необработанным заявкам
